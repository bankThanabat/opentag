# Platform Setup Guides

OpenTag can listen to different work platforms and route each request to the same local coding-agent runtime.

Use these guides when `opentag setup` asks for platform credentials:

- [Lark / Feishu](lark.en.md) / [简体中文](lark.zh-CN.md)
- [Slack](slack.en.md) / [简体中文](slack.zh-CN.md)
- [GitHub](github.en.md) / [简体中文](github.zh-CN.md)

Each guide only covers the values the OpenTag CLI asks for. For deeper integration debugging, see [Real integration smoke test](../real-integration-smoke-test.md).
