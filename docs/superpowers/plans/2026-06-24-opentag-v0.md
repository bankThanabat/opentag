# OpenTag V0 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the first working OpenTag vertical slice: a GitHub-style mention becomes an auditable run, a local daemon claims it, an echo executor runs it, and a callback summary is produced.

**Architecture:** Use a TypeScript pnpm monorepo with dependency-free core schemas, a thin dispatcher backed by SQLite/Drizzle, a Probot GitHub ingress app, and a local `opentagd` daemon that polls and executes runs. Keep framework dependencies at the edges so `opentag-core` remains portable.

**Tech Stack:** TypeScript, pnpm workspaces, tsup, Vitest, Zod, Hono, Drizzle ORM, SQLite/libSQL, Probot, Octokit, Commander.

---

## Defaults Locked For V0

- Implementation language: TypeScript only.
- Dispatcher: hosted HTTP service with SQLite-compatible persistence.
- Runner communication: polling plus leases, not inbound local webhooks.
- First executor: `echo`, then Codex/Claude Code/oh-my-pi adapters after the loop works.
- GitHub ingress: Probot at the edge.
- Core schema: Zod-first, JSON Schema export later.
- Storage: Drizzle schema over SQLite/libSQL.
- Write operations: v0 posts comments only; PR creation is Milestone 4.

- [ ] **Step 7: Implement mention parser**

Create `packages/core/src/mention.ts`:

```ts
import type { OpenTagCommand } from "./schema";

type MentionMatch = { matched: false } | ({ matched: true } & OpenTagCommand);

const INTENTS: OpenTagCommand["intent"][] = ["fix", "review", "investigate", "explain", "run"];

export function parseOpenTagMention(body: string, mention = "@opentag"): MentionMatch {
  const escaped = mention.replace(/[.*+?^${}()|[\]\\]/g, "\\$&");
  const pattern = new RegExp(`${escaped}\\s+([^\\n\\r]+)`, "i");
  const match = body.match(pattern);

  if (!match?.[1]) {
    return { matched: false };
  }

  const rawText = match[1].trim();
  const firstWord = rawText.split(/\s+/, 1)[0]?.toLowerCase();
  const intent = INTENTS.includes(firstWord as OpenTagCommand["intent"])
    ? (firstWord as OpenTagCommand["intent"])
    : "unknown";

  return {
    matched: true,
    rawText,
    intent,
    args: {}
  };
}
```

Create `packages/store/src/repository.ts`:

```ts
import type { OpenTagEvent, OpenTagRun, OpenTagRunResult } from "@opentag/core";
import { eq } from "drizzle-orm";
import type { BetterSQLite3Database } from "drizzle-orm/better-sqlite3";
import { repoBindings, runEvents, runners, runs } from "./schema";

function nowIso(): string {
  return new Date().toISOString();
}

export function createOpenTagRepository(db: BetterSQLite3Database) {
  return {
    async registerRunner(input: { runnerId: string; name: string }): Promise<void> {
      const createdAt = nowIso();
      await db.insert(runners).values({ runnerId: input.runnerId, name: input.name, createdAt }).onConflictDoNothing();
    },

    async createRepoBinding(input: { provider: string; owner: string; repo: string; runnerId: string }): Promise<void> {
      await db.insert(repoBindings).values({ ...input, createdAt: nowIso() });
    },

    async createRun(input: { id: string; event: OpenTagEvent }): Promise<OpenTagRun> {
      const createdAt = nowIso();
      await db.insert(runs).values({
        id: input.id,
        eventId: input.event.id,
        status: "queued",
        eventJson: JSON.stringify(input.event),
        createdAt,
        updatedAt: createdAt
      });
      await db.insert(runEvents).values({
        runId: input.id,
        type: "run.created",
        payloadJson: JSON.stringify({ eventId: input.event.id }),
        createdAt
      });
      return {
        id: input.id,
        eventId: input.event.id,
        status: "queued",
        createdAt,
        updatedAt: createdAt
      };
    },

    async claimNextRun(input: { runnerId: string; leaseSeconds: number }): Promise<OpenTagRun | null> {
      const row = await db.select().from(runs).where(eq(runs.status, "queued")).limit(1).get();
      if (!row) return null;

      const updatedAt = nowIso();
      const leaseExpiresAt = new Date(Date.now() + input.leaseSeconds * 1000).toISOString();
      await db
        .update(runs)
        .set({
          status: "assigned",
          assignedRunnerId: input.runnerId,
          leaseExpiresAt,
          updatedAt
        })
        .where(eq(runs.id, row.id));
      await db.insert(runEvents).values({
        runId: row.id,
        type: "run.claimed",
        payloadJson: JSON.stringify({ runnerId: input.runnerId, leaseExpiresAt }),
        createdAt: updatedAt
      });

      return {
        id: row.id,
        eventId: row.eventId,
        status: "assigned",
        assignedRunnerId: input.runnerId,
        executor: row.executor ?? undefined,
        createdAt: row.createdAt,
        updatedAt
      };
    },

    async markRunning(input: { runId: string; executor: string }): Promise<void> {
      const updatedAt = nowIso();
      await db.update(runs).set({ status: "running", executor: input.executor, updatedAt }).where(eq(runs.id, input.runId));
      await db.insert(runEvents).values({
        runId: input.runId,
        type: "run.running",
        payloadJson: JSON.stringify({ executor: input.executor }),
        createdAt: updatedAt
      });
    },

    async completeRun(input: { runId: string; result: OpenTagRunResult }): Promise<void> {
      const updatedAt = nowIso();
      const status = input.result.conclusion === "success" ? "succeeded" : "failed";
      await db
        .update(runs)
        .set({ status, resultJson: JSON.stringify(input.result), updatedAt })
        .where(eq(runs.id, input.runId));
      await db.insert(runEvents).values({
        runId: input.runId,
        type: "run.completed",
        payloadJson: JSON.stringify(input.result),
        createdAt: updatedAt
      });
    }
  };
}

export function renderFinalResult(result: OpenTagRunResult): string {
  const lines = [`OpenTag finished with **${result.conclusion}**.`, "", result.summary];
  if (result.verification?.length) {
    lines.push("", "Verification:");
    for (const check of result.verification) {
      lines.push(`- \`${check.command}\`: ${check.outcome}`);
    }

import { OpenTagEventSchema, OpenTagRunResultSchema } from "@opentag/core";
import { createOpenTagRepository, migrateSchema } from "@opentag/store";
import Database from "better-sqlite3";
import { drizzle } from "drizzle-orm/better-sqlite3";
import { Hono } from "hono";
import { z } from "zod";

const CreateRunnerSchema = z.object({
  runnerId: z.string().min(1),
  name: z.string().min(1)
});

const CreateRunSchema = z.object({
  runId: z.string().min(1),
  event: OpenTagEventSchema
});

const CompleteRunSchema = z.object({
  result: OpenTagRunResultSchema
});

export function createDispatcherApp(input: { databasePath: string }) {
  const sqlite = new Database(input.databasePath);
  migrateSchema(sqlite);
  const repo = createOpenTagRepository(drizzle(sqlite));
  const app = new Hono();

  app.post("/v1/runners", async (c) => {
    const parsed = CreateRunnerSchema.parse(await c.req.json());
    await repo.registerRunner(parsed);
    return c.json({ ok: true }, 201);
  });

  app.post("/v1/runs", async (c) => {
    const parsed = CreateRunSchema.parse(await c.req.json());
    const run = await repo.createRun({ id: parsed.runId, event: parsed.event });
    return c.json({ run }, 201);
  });

  app.post("/v1/runners/:runnerId/claim", async (c) => {
    const run = await repo.claimNextRun({ runnerId: c.req.param("runnerId"), leaseSeconds: 60 });
    if (!run) return c.json({ run: null }, 204);
    return c.json({ run }, 200);
  });

  app.post("/v1/runs/:runId/running", async (c) => {
    const body = z.object({ executor: z.string().min(1) }).parse(await c.req.json());
    await repo.markRunning({ runId: c.req.param("runId"), executor: body.executor });
    return c.json({ ok: true });
  });

  app.post("/v1/runs/:runId/complete", async (c) => {
    const parsed = CompleteRunSchema.parse(await c.req.json());
    await repo.completeRun({ runId: c.req.param("runId"), result: parsed.result });
    return c.json({ ok: true });
  });

  return app;
}
```

export async function runOneDaemonIteration(input: {

    {

      workspacePath: input.workspacePath,
      command: claimed.command,
      context: claimed.context
    },
    {
      emit: async (event) => {
        console.log(`[${event.type}] ${event.message}`);
      }
    }
  );
  await input.client.complete(claimed.run.id, result);
  return true;
}

import type { DaemonClient } from "./daemon";

export function createDispatcherClient(input: { dispatcherUrl: string; runnerId: string }): DaemonClient {
  const baseUrl = input.dispatcherUrl.replace(/\/$/, "");

  return {
    async claim() {
      const response = await fetch(`${baseUrl}/v1/runners/${input.runnerId}/claim`, { method: "POST" });
      if (response.status === 204) return null;
      if (!response.ok) throw new Error(`claim failed: ${response.status}`);
      const body = (await response.json()) as { run: { id: string; eventId: string; status: "assigned"; createdAt: string; updatedAt: string }; event?: unknown };
      if (!body.run) return null;
      return {
        run: body.run,
        command: { rawText: "echo run", intent: "run", args: {} },
        context: []
      };
    },
    async markRunning(runId, executor) {
      await fetch(`${baseUrl}/v1/runs/${runId}/running`, {
        method: "POST",
        headers: { "content-type": "application/json" },
        body: JSON.stringify({ executor })
      });
    },
    async complete(runId, result) {
      await fetch(`${baseUrl}/v1/runs/${runId}/complete`, {
        method: "POST",
        headers: { "content-type": "application/json" },
        body: JSON.stringify({ result })
      });
    }
  };
}
```

import { normalizeGitHubIssueComment, renderAcknowledgement } from "@opentag/github";
import type { Probot } from "probot";

type IssueCommentPayload = {
  comment: { id: number; body: string; html_url: string };
  issue: { html_url: string; comments_url: string };
  repository: { name: string; private: boolean; owner: { login: string } };
  sender: { id: number; login: string };
};

export async function handleIssueCommentCreated(input: {
  payload: IssueCommentPayload;
  createRun(event: unknown): Promise<{ runId: string }>;
  postComment(body: string): Promise<void>;
  now(): string;
}): Promise<void> {
  const event = normalizeGitHubIssueComment({
    id: String(input.payload.comment.id),
    commentBody: input.payload.comment.body,
    commentUrl: input.payload.comment.html_url,
    apiCommentsUrl: input.payload.issue.comments_url,
    issueUrl: input.payload.issue.html_url,
    owner: input.payload.repository.owner.login,
    repo: input.payload.repository.name,
    actorId: input.payload.sender.id,
    actorLogin: input.payload.sender.login,
    private: input.payload.repository.private,
    receivedAt: input.now()
  });

  if (!event) return;

  const { runId } = await input.createRun(event);
  await input.postComment(renderAcknowledgement(runId));
}

export function createOpenTagProbotApp(app: Probot): void {
  app.on("issue_comment.created", async (context) => {
    await handleIssueCommentCreated({
      payload: context.payload as IssueCommentPayload,
      createRun: async () => {
        const runId = `run_${Date.now()}`;
        return { runId };
      },
      postComment: async (body) => {
        await context.octokit.issues.createComment(context.issue({ body }));
      },
      now: () => new Date().toISOString()
    });
  });
}
```

# OpenTag

OpenTag is the open mention layer for agents.

Tag Codex, Claude Code, Pi, or your own local runner from GitHub, Slack, or Lark. OpenTag turns the mention into an auditable run, dispatches it to an approved local or hosted executor, and reports the result back to the original workspace.

## Why

OpenTag brings approved agents into the workspace where the request already has context.

## V0 Direction

The first implementation focuses on a narrow GitHub-to-local-runner loop:

1. A GitHub comment mentions `@opentag`.
2. A Probot app normalizes the event.
3. A thin hosted dispatcher stores and leases the run.
4. A local daemon claims the run.
5. An executor adapter runs the task.
6. OpenTag reports the result back to GitHub.

## Packages

- `packages/core`: protocol schemas and mention parsing.
- `packages/github`: GitHub event normalization and callback rendering.
- `packages/store`: SQLite/Drizzle persistence and lease primitives.
- `packages/runner`: executor contracts and the echo executor.
- `apps/dispatcher`: hosted dispatcher API.
- `apps/opentagd`: local runner daemon.
- `apps/github-probot`: GitHub App ingress.

## Commands

```bash

pnpm build
```

## Design

See [`docs/design.md`](docs/design.md).
